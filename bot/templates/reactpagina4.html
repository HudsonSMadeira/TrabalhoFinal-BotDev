O que é JSX e como ele facilita a criação de interfaces de usuário no React?

Definição: JSX (JavaScript XML) é uma sintaxe semelhante a HTML para JavaScript usada no React, que facilita a criação de interfaces de usuário ao permitir que HTML e JavaScript sejam escritos juntos. Isso torna o código mais intuitivo e permite manipulação dinâmica do conteúdo com expressões JavaScript. JSX simplifica a manutenção e evolução do código, promovendo a reutilização de componentes e uma construção mais eficiente de interfaces dinâmicas.
