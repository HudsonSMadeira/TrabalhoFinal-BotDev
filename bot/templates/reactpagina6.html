O que é o Virtual DOM e como ele otimiza a atualização da interface no React?

Definição:É uma representação virtual da estrutura hierárquica da interface de usuário em memória, mantida pelo React. Ele funciona como uma cópia leve e eficiente do DOM real do navegador. Quando ocorre uma atualização de estado em um componente React, o Virtual DOM é comparado com o estado anterior para identificar as diferenças (diffing). Com base nessas diferenças, o React determina os mínimos conjuntos de alterações necessários no DOM real para refletir essas mudanças.
